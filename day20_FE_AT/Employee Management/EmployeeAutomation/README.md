# Employee Management System - Automation Testing

## Project Overview

This project focuses on automation testing of a React-based Employee Management System using Selenium WebDriver and JUnit.

The application allows users to manage employee records including adding, editing, deleting, searching and viewing employee details.

---

## Technology Stack

- Java
- Selenium WebDriver
- JUnit 5
- Maven
- ChromeDriver
- VS Code

The following modules are automated:

1. Login/Home Page
2. Navigation
3. Dashboard
4. Employees Module
5. Search Functionality
6. Add Employee
7. Edit Employee
8. Delete Employee
9. Form Validation
10. Activity Section

---

## Test Cases Covered

### Login Testing

- Verify application opens successfully
- Verify page title
- Verify WorkSphere heading
- Verify login button

### Employee Testing

- Verify employee table display
- Verify search functionality
- Verify add employee
- Verify edit employee
- Verify delete employee

### Validation Testing

- Empty form validation
- Invalid name validation
- Invalid email validation
- Invalid phone validation
- Invalid salary validation
